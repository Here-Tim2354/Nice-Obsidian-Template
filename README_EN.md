# Template Vault

This is a small Obsidian theme and plugin bundle by Tim2354. It is designed to be used directly as a template vault. It includes a set of frequently used plugins and a ready-made visual theme. You can also build on top of this template and customize it into a setup that better fits your own workflow.

## Quick Start

Copy the files from this project, usually just `.obsidian/*`, to any folder you want. This can be a documentation folder inside a project, or your personal note vault.

Open that folder in Obsidian, choose to trust the vault, and enable the plugins.

If you are an English user, you may also need to switch the language of Obsidian.

Note: If you find that a `.obsidian` folder already exists in a directory, it means you have previously created an Obsidian Vault there. It is recommended to create a new Vault in a directory that does not contain a `.obsidian` folder.

## Appearance

- Theme: Border
- Accent color: `#e48b25`
- UI font: [HONOR Sans CN](https://developer.honor.com/cn/doc/guides/100681)
- Text font: Inter -> [MiSans](https://hyperos.mi.com/font/zh/)
- Monospace font: [JetBrains Mono](https://www.jetbrains.com/lp/mono/) -> HONOR Sans CN
- Base font size: 18
- Workspace: use `Default` to get the author's recommended layout

Note: Fonts are not bundled with this template. They are only the author's recommendations. All listed fonts are commercially usable and can be downloaded from their official websites.

## Plugins

### Community Plugins

| Plugin | Purpose |
| --- | --- |
| Calendar | Adds a calendar view for daily notes |
| File Explorer Note Count | Shows the number of notes inside folders in the file explorer |
| Export Image plugin | Exports a note or selected content as an image |
| Importer | Imports notes from other sources |
| Pandoc Plugin | Exports notes to DOCX, ePub, PDF, and more via Pandoc |
| Missing Link File Creator | Creates files for missing wiki links |
| Simple Code Formatter | Formats the current code block |
| Style Settings | Adjusts theme, plugin, and CSS variables |
| Templater | Provides advanced templates and automation |
| Iconize | Adds icons to files, folders, and text |
| Terminal | Opens a terminal inside Obsidian |
| markdown export | Exports Markdown files |
| Easy Typing | Improves Chinese and English editing behavior |
| Enhancing Export | Enhanced export support for HTML, DOCX, PDF, PPTX, and more |
| Git | Provides a Git GUI inside Obsidian |
| Neighbouring Files | Navigates between files in the same folder |

Most plugin features can be accessed from the command palette.

### Core Plugins

File explorer, search, graph view, backlinks, Canvas, tags, properties, page preview, daily notes, templates, note composer, command palette, bookmarks, outline, slides, and most other core plugins are enabled.

Disabled core plugins include Slash Commands, Markdown Importer, ZK Prefixer, Random Note, Publish, and Sync.

## More Settings

- **Trash**: local `.trash`
- **New file location**: the folder of the current page. If there is no current page, files are created in the vault root.
- **Attachment path**: `Attachment/`. For example, pasted images are saved to this folder by default.
- **Always update links**: enabled, so moving notes will not break links.
- **Spell check**: disabled.
- **Line numbers**: disabled. Indentation guides are enabled.
- **Delete confirmation**: **disabled**. Deleting files does not require confirmation.
- **Typing enhancement**: includes a series of settings optimized for Chinese input. See the `Easy Typing` plugin settings for details.
