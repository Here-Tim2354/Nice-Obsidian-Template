# Template Vault

[English](README_EN.md)

这是Tim2354的一个 Obsidian 主题和插件的小整合包，适合作为模板直接使用。内置了使用频率较高的插件，和一套现成的外观主题。同时也完全可以基于本模版二次开发为更适合自己的主题。

## 快速开始

复制本项目的文件（`.obsidian/*`即可）到你的任何地方，无论是某个项目的文档文件夹下，还是你的私人笔记库。

打开Obsidian到你指定的文件夹下，选择信任并启用插件即可完成。

## 外观

- 主题：Border
- 强调色：`#e48b25`
- UI 字体：[HONOR Sans CN](https://developer.honor.com/cn/doc/guides/100681)
- 正文：Inter → [MiSans](https://hyperos.mi.com/font/zh/)
- 等宽：[JetBrains Mono](https://www.jetbrains.com/lp/mono/) → HONOR Sans CN
- 基础字号：18
- 工作区：可使用`默认`来获取作者推荐布局

注意：字体并不包含在内，只是作者推荐。均为可商用字体，可通过官方网站下载。

### 核心插件

文件管理、搜索、图谱、反链、Canvas、标签面板、属性、页面预览、日记、模板、笔记重组、命令面板、书签、大纲、幻灯片……基本全开。

关闭的有：斜杠命令、Markdown 导入、ZK 前缀、随机笔记、发布、Sync。

## 更多配置

- **回收站**：本地`.trash`
- **新建文件位置**：当前页面所在目录下。若空白，则为根目录。
- **附件路径**：`Attachment/`。比如复制的图片会默认进入这个文件夹。
- **始终更新链接**：开——移动笔记不会断链
- **拼写检查**：关
- **行号**：关，缩进线：开
- **删除确认**：**关**。即删除文件不需要你确认。
- **输入增强**：含有一系列针对中文输入优化的配置。具体查看`Easy Typing`的配置。
